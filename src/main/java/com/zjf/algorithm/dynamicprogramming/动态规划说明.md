首先，动态规划问题的一般形式就是求最值。动态规划其实是运筹学的一种最优化方法，
只不过在计算机问题上应用比较多，比如说让你求最长递增子序列呀，最小编辑距离呀等等


动态规划问题最困难的就是写出这个暴力解，即状态转移方程

寻找状态之间的关系 ->确定 base case(基础值)
